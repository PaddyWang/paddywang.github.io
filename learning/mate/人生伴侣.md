## 婚姻的幸福度
* 单身 -> ~~缘分~~ -> 结婚
* 不好的婚姻 远不如单身的幸福

## 选择伴侣是最重要的选择 <br> <span class="text-[12px]">并且完全是由自己选择</span>
* 陪伴最久的人(90岁 - 现在的年龄)
* 2万多个平凡的每一天组成

## 伴侣对我们的影响 <div class="text-[12px]">会渗透到我们生活中每个细节</div>
* 一起生活最久的人
* 一起养育孩子的人
* 一起吃6万顿饭的人 睡2万多天的人
* 一起成长的人
* 一起过年回家的人
* 一起给父母养老的人

## 影响选择逆反因子
* 自己不清楚到底想要什么
* 单身和结婚的需求完全不一样
  - **转换时容易迷失自我**
* 没有试错成本
* 太相信缘分了
* 很大可能会选择圈子内相似的人
  - 扩大自己的圈子
  - 可以网络认识
    + 大家对于网上认识的人会感觉不正统 有偏见
* 婚姻截止日期
  - 社会 父母 催着我们结婚
* 生理“捣乱”
  - 甜蜜的陷阱
    + 心动 -> 荷尔蒙
      * 上头 失去判断力
  - 身体催促(年龄到了)

## 婚姻关系受害者
* 浪漫宿醉者(爱情是全部)
  + 忽略了生活中琐事的摩擦
  + 爱情自我催眠
    * 不断的告诉自己当初在一起多不容易
      - 压抑自己内心的不安
* 担惊受怕者(年龄焦虑) <div class="text-[12px]">让我们恐惧的应该是**和不合适的人共度余生**</div>
  + 害怕自己剩下
  + 害怕自己被议论
  + 害怕自己。。。
* 肤浅者
  - 太在意别人的看法
  - 缺少心灵共鸣
* 自私者
  - 自己的需求永远第一 <div class="text-[12px]">会把夫妻之间的平等 幸福消耗掉</div>
    + 不愿意妥协 牺牲
    + 一个人过日子
  - 新鲜感一旦过去只剩空虚
    + 最终会变得死寂

## 受害者共性问题
* 没有搞清楚现实生活的婚姻是什么样
* 他们理想中爱人的样子都是错的

## 幸福的婚姻
* 有无数个普通的小的瞬间组成
  - 每天的相处
* 基础：真挚的友情
  - 开心 有说不完的话 愿意一起折腾 愿意待在一起
  - 交通压力测试
    + 真实的幽默感
    + 有趣
    + 乐观
  - 互相理解 互相信任
  - 尊重彼此的思维方式和生活方式
    + 愿意听对方的想法和反馈
* 家的感觉 <br> **比激情更重要的是两个人自在的相处**
  - 安全 自在 放松 不需要伪装自己 完全做自己 氛围温暖舒服  
  - 需要相处的舒服
    + 不然小的摩擦就会被无限放大
      * 最后变成无法忍受的负担
  - 需要建立家的感觉
    + 信任
      * 秘密会让彼此产生隔阂 腐蚀掉安全感
    + 沟通同频 <br> 能量水平差不多
    + 接纳对方的缺点
      * 不是去改造而是接纳
    + 营造积极的氛围
* 如何去经营美好的婚姻？
  - 基础：**沟通**
  - 遇到分歧
    + 技巧
      * 幽默 温和 -> 化解紧张感
      * 不带滤镜的去倾听对方真实的想法
    + 有69%的矛盾没办法解决 -> 学会绕开无解的问题

## 评估/审视一段关系 <div class="text-[12px]">每段关系都有短板</div>
* 影响日常相处的核心东西
  - 能否有效沟通
  - 能否彼此信任
  - 关键问题能否达成共识
* 兴趣爱好 小习惯
  - 不太重要
* 不是靠感觉 运气 缘分
  - **一定要看清自己内心**
* 能否幸福走下去
  - 普通的日子 + **深度连接**

## 灵魂伴侣 需要具备“反直觉”特质 <div class="text-[12px]">不要一味的寻找</div>
* 差异共生
  - **不是填补人生的空缺** 不是破船找补丁
  - **能享受独处** 完整的人才能遇到另一个完整
* 孕者永生
  - 不是遇到主动出击 不要在意八字 星座
  - **创造自己的高价值能量场域(磁场)**
  - 独立的完整才能创造出来合一的完整
* 破镜成门
  - 三观一致是悖论
  - 那些拍案而起的争吵 都是宇宙在提醒你：你触碰到了自己的认知边界
  - 鲁米说过：伤口才是光进入你内心的地方
  - 运用荣格的：阴影自我 -> 阴影工作
:::


## 总结
* 选择伴侣是我们一生最重要的决定
* 灵魂伴侣是极难寻找的 要“反直觉”
* 伴侣不是填补人生的空缺
* 创造自己的高价值能量场域
* 能享受独处
* 单身 -> 婚姻 很容易迷失自我
* 要恐惧和不合适的共度余生
* 要搞清楚现实生活中婚姻的样子
* 比激情更重要的是两个人自在的相处
* 沟通最重要
* 在一段关系中：一定要看清自己内心
* 在一段关系中：要 深度连接
