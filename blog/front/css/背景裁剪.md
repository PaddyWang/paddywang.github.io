# background-clip
## border-box
为默认值。背景会被沿着边框外沿进行剪切，即会延伸到边框下面  

:::codeview
---
lang: html{10}
shadow: true
---
<style>
  .background-clip-border-box {
    height: 200px;
    border: 10px dashed #000;
    padding: 20px;
    font-size: 36px;
    text-align: center;
    background-image: url(/image.png);
    background-size: contain;
    background-clip: border-box;
  }
</style>
<div class="background-clip-border-box">这里是文本内容</div>
:::

## text
背景会被裁剪成文本的前景色  
该属性为实验性属性，慎用，目前需要浏览器前缀  
:::codeview
---
lang: html{10,11}
shadow: true
---
<style>
  .background-clip-text {
    height: 200px;
    border: 10px dashed #000;
    padding: 20px;
    font-size: 36px;
    text-align: center;
    background-image: url(/image.png);
    background-size: contain;
    background-clip: text;
    -webkit-background-clip: text;
    color: transparent;
    line-height: 200px;
  }
</style>
<div class="background-clip-text">这里是文本内容</div>
:::
